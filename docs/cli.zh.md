## 命令行

全局选项

- config: 项目配置文件路径，默认 `./.htte.yaml`

### run

运行测试, 为默认命令，`htte` 等同于 `htte run`.

选项:

- debug: 打印请求和响应数据　
- amend: 从上次中断的地方开始运行测试
- bail: 测试不通过，中断后续测试
- unit: 从指定的单元测试处开始运行测试
- shot: 执行单个测试，不再运行后续测试 

### inspect

审视特定单元测试，显示该测试相关的所以信息 


### view

查看测试, 可以看到所有的测试及其组织架构。

选项:

- module: 仅显示指定模块下的测试
- api: 仅显示指定 api 的测试
